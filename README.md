# Menufit

**Menufit** — Cuisine française/européenne pour débutants. Local-first, zéro pub, zéro friction.

## Principes

- 🏠 **Local-first absolu** — fonctionne entièrement hors ligne
- 🧠 **Zéro friction cognitive** — interface simple et respectueuse
- 💪 **Respect total** — pas de pub, pas de dark patterns
